
# Digital Quiz Demo - Vercel Ready

هذه نسخة جاهزة لنشر كويز التسويق الرقمي على Vercel.
يمكنك رفع هذا المشروع مباشرة على GitHub وربطه بـ Vercel.

## التعليمات السريعة
1. افتح Terminal أو PowerShell في المجلد بعد فك الضغط.
2. تثبيت التبعيات:
   ```bash
   npm install
   ```
3. تشغيل المشروع محلياً:
   ```bash
   npm start
   ```
4. رفع المشروع على GitHub ثم نشره على Vercel.
